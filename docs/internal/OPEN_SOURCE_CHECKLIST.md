# Open-Source Checklist (NeurIPS-oriented)

- [ ] One-command environment setup (`pip install -e .` or `conda env create`)
- [ ] One-command minimal smoke run (tiny config)
- [ ] Full training command(s) for main results
- [ ] Evaluation command(s) and metric extraction command(s)
- [ ] Exact random seed policy documented
- [ ] Hardware requirement and expected runtime documented
- [ ] Data access instructions + license constraints documented
- [ ] Model checkpoints publication plan documented
- [ ] Unit tests for critical utilities
- [ ] CI to run lint + tests
- [ ] `CITATION.cff` and BibTeX entry included
- [ ] Clear license for code and third-party dependencies
